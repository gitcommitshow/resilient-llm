# Changelog

## Dec 31, 2025

### Added
- **Regenerate assistant messages** - Regenerate any assistant response with a single click
- **Copy message** - Copy any message content to clipboard
- **Click outside to close** settings drawer
- **Keyboard shortcuts** - `Escape` to cancel, `Enter` to save settings
- **Status bar** - Quick access to settings from chat panel header

### Changed
- **Visual design** - Modern shadow-based UI (replaced borders)
- **Settings save** - Now saves when drawer closes instead of while editing

### Fixed
- **Message ordering** - Fixed bug where messages could appear in wrong order
